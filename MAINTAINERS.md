# Maintainers

Machine-readable governance metadata for the Civic Influence Graph project
(Constitution III — Transparency).

## Format

Each maintainer entry includes:
- **name**: Display name
- **github**: GitHub username
- **role**: `lead` | `maintainer` | `contributor`
- **scope**: Area(s) of responsibility
- **since**: Date joined as maintainer (YYYY-MM-DD)

## Governance

- **Decision process**: Consensus among active maintainers; lead breaks ties.
- **Constitutional changes**: Require approval from all active maintainers with `lead` or `maintainer` role.
- **New maintainers**: Nominated by existing maintainer, approved by lead.
- **Removal**: Inactivity > 6 months or consensus vote.
